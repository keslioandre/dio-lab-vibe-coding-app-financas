# 💸 App de Organização de Finanças Pessoais com Vibe Coding -  Seu Agente Financeiro

PRD refinado no Copilot Web:
``` markdown
PRD – Aplicativo de Organização de Finanças Pessoais

#Contexto
Criar um aplicativo que ajude pessoas a organizar suas finanças pessoais por meio de conversas em linguagem natural.
O objetivo é simplificar o controle financeiro, eliminando formulários complexos e planilhas manuais, oferecendo uma experiência fluida e acessível.

#Problema
Muitos usuários desistem de controlar seus gastos porque os aplicativos atuais exigem:
- Entrada manual excessiva
- Pouca personalização
- Experiência pouco intuitiva

#Solução proposta: um app que funcione como um “Agente Financeiro” conversacional, capaz de registrar, classificar e sugerir melhorias de forma automática, com design universal para garantir que o máximo de pessoas possa usar com boa experiência.

#Público-Alvo
- Pessoas que desejam começar a organizar suas finanças sem complicação
- Usuários iniciantes em controle financeiro
- Pessoas que preferem interações simples e naturais em vez de interfaces tradicionais
- Diversos perfis de usuários, incluindo aqueles com diferentes níveis de letramento digital ou necessidades de acessibilidade

#Funcionalidades-Chave
1. Registro de gastos via chat em linguagem natural
2. Classificação automática das transações (ex.: alimentação, transporte, lazer)
3. Definição e acompanhamento de metas financeiras (ex.: economizar R$500/mês)
4. Dicas de economia personalizadas fornecidas pelo “Agente Financeiro”
5. Relatórios simples e visuais adaptados ao perfil do usuário
6. Design universal: interface inclusiva, acessível e intuitiva para diferentes públicos (ex.: suporte a leitores de tela, contraste adequado, linguagem clara)

#Entregável da IA
- Plano de MVP com:
  - Principais telas (chat, metas, relatórios)
  - Recursos necessários (NLP para entender linguagem natural, categorização automática, motor de recomendações)
  - Esboço de validação inicial (testar com grupo diverso de usuários, incluindo iniciantes e pessoas com necessidades de acessibilidade)
- Linguagem acessível e educativa, em português
```

## 💬 Conclusão

Vibe Coding é sobre clareza, curiosidade e criatividade, não sobre perfeição técnica. O verdadeiro objetivo aqui é aprender a pensar junto com a IA, transformando ideias em conceitos reais e enxergando a tecnologia como uma extensão do seu raciocínio criativo. Cada interação é um experimento, quanto mais clara for sua intenção, mais surpreendente será o resultado.
